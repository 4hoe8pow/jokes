## 概要
**Dirty Socks** は、汚れた Socks たちを各「現場」から脱出させるスライドパズル型ローグライクです。
プレイヤーは Socks を操作してグラフ状の現場を探索し、脱出ハッチ到達を目指します。
ゲーム進行は **ネットワークグラフ** と **テキストログ** を中心に展開された可視性制御が特徴です。

---

## ゲーム目的
- 4 枚の Socks のうち **1 枚でも脱出ハッチに到達** すれば現場クリア
- すべての Socks が力尽きるとゲームオーバー
- 各現場には **想定手数（PAR）** が設定され、実手数との差分でスコア算出（ゴルフ方式）
- クリア後は次の現場へ進行し、ステータスは継続（回復なし）
- より多くの現場を脱出し、累積スコアを最小化する

---

## プレイヤーアクション
1. **移動操作**
   - 任意の Socks を選び、隣接ノードへスライド移動。
   - バッジは互いにすれ違えず、単線構造がパズル制約となる。

2. **イベント発生**
   - ノード到達時、その種類に応じたイベントが自動発生し、テキストログに表示。
   - イベント中の Socks はロックされ、操作不可。

3. **並列性**
   - ほかの Socks がイベント処理中でも、残りは操作可能。
   - 全員がイベント中の場合のみ待機状態となる。

---

## ノードの種類
- **スタート**：プレイヤー投入地点。
- **ハッチ**：脱出地点。到達すればクリア。
- **敵**：自動戦闘が発生。勝敗はステータス比較で決定。勝利すれば再操作可能、敗北すると消滅。
- **ヨゴレ溜まり**：ダーティネス上昇。強化を得られるが UI 可視性が低下。
- **浄化罠**：ダーティネス減少。UI がクリアになるが強化を失う。
- **転移**：ランダムに別ノードへワープ。計画を乱すが有利な位置に出る可能性も。
- **ミニドラマ**：状態依存の演出分岐。ダーティネスや残存数によって特殊イベントやホラー演出が発動。

  ```swift
  enum NodeKind {
      case start        // 出発点:必ず４箇所
      case hatch        // ハッチ:必ず１箇所
      case enemy        // 敵（洗濯バサミなど）
      case dirtPool     // 汚れ溜まり（ダーティネス回復・強化）
      case purification // 浄化罠（ダーティネス減少・ペナルティ）
      case warp         // 転移（他のノードへランダム転移）
      case skit    // 寸劇（ステータス依存イベント、一部ホラー）
  }
  ```

## ダーティネスシステム
- **バフ**：敵への強さ補正。
- **副作用**：UI が濁り、数値やログがマスク・グリッチ化。
- **限界状態**：極限に達すると画面は真っ黒化し、操作困難に。脱出までが時間との勝負となる。

---

## UI/UX ガイドライン
| 課題               | 解決策                                                                               |
| ------------------ | ------------------------------------------------------------------------------------ |
| 情報過多           | ネットワークグラフとログを基盤に整理。アイコンでイベント種別のみ表示し、詳細は到達時に開示。 |
| 行動の影響が不明瞭 | イベント結果や環境変化をログで即時表示。差分を明確化。                               |
| 戦略学習が難しい   | 行動履歴を記録し、振り返り可能。試行錯誤の蓄積を支援。                               |
| 可視性の低下       | 危険やダーティネス変化を色・フォントで強調。                                         |
| 乱数による理不尽感  | 確率要素は転移や一部演出のみ。発生確率を明示し、結果をログ化。                       |

## 隠し要素
- 特定のノード通過パターンやダーティネス条件により、ホラー演出や特殊恩恵が発生。
- 通常プレイでは気づきにくい「裏コマンド」的要素を仕込み、探索性とリプレイ性を高める。
